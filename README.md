# ИИ-трекер рабочего времени

Десктопное приложение для фонового трекинга активности, контроля отвлекающих факторов и управления задачами с ИИ-ассистентом.

## Функции
- Фоновый трекинг окон и игр
- Статистика использования приложений
- Список дел с подзадачами и прогрессом
- ИИ-чат (GigaChat)
- Уведомления с миганием светодиода на ESP8266

## Установка
1. Установите Python 3.12+
2. Установите зависимости, введя в консоль команду: `pip install -r requirements.txt`
3. Запустите: `python tracker_with_notifications.py`

## Сборка EXE
Введите в консоль: `pyinstaller --onefile --noconsole --name="AITimeTracker" --add-data "assets/icon.ico;assets" tracker_with_notifications.py`
